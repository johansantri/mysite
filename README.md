# mysite

Quickstart
---------
clone & Run mysite

1. `sudo git clone https://github.com/johansantri/mysite.git`
2. `cd mysite`
3. `python3 -m venv .venv`
4. `pip freeze > requirements.txt`
5. `python3 manage.py runserver`
