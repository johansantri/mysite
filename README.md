# mysite

Quickstart
---------
clone & Run mysite
1. `sudo apt install git`
2. `mkdir app`
3. `cd app`
4. `sudo git clone https://github.com/johansantri/mysite.git`
5. `python3 -m pip install Django`
6. `pip install django-ckeditor`
7. `pip install django-taggit`
8. `python3 manage.py runserver`
