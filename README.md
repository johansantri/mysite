# mysite

Quickstart
----------
1. `clone & Run mysite`
2. `sudo apt install git`
3. `mkdir app`
4. `cd app`
5. `sudo git clone https://github.com/johansantri/mysite.git`
6. `python3 -m pip install Django`
7. `pip install django-ckeditor`
8. `pip install django-taggit`
9. `sudo chmod 777 mysite`
10. `python3 manage.py runserver`
